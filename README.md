# Pantry-To-Plate
Pantry To Plate (P2P) is a Streamlit-based web application that helps users discover recipes based on the ingredients they already have at home.
Instead of searching for a dish first, users simply enter the available ingredients, and P2P recommends recipes that can be prepared using those ingredients.

For each recommended recipe, the application provides comprehensive information, including:

1. Preparation and Cooking Time
2. Flavor Profile (e.g., spicy, sweet, savory, tangy)
3. Nutritional Information (per serving)
4. YouTube Video Tutorial for step-by-step cooking guidance
5. Macro and Micronutrient Distribution, helping users make informed dietary choices

Pantry To Plate aims to reduce food waste, simplify meal planning, and encourage healthier eating by making the most of the ingredients already available in a user's pantry.
